## ðŸ”® Usage

Run development environment:

```bash
# Install packages
$ yarn
# Up docker compose environment
$ docker-compose up
# Setup dotenv variables
$ cp .env.example .env
# Launch app
$ yarn dev
```

Build & Run production environment:

```bash
# Install packages
$ yarn
# Build project
$ yarn build
# Launch app through Yarn
$ yarn start
# Launch app through Node
$ node dist/main.js
```
